# **App Name**: Gratitude Garden

## Core Features:

- Daily Entry: Users can create a new gratitude entry each day.
- Daily Prompt: Provide a thoughtful prompt to guide daily reflection.
- Entry Browsing: Browse past entries, sorted by date.
- Mood Tracker: Record the user's mood along with each entry.
- AI Mood Analysis Tool: The application will suggest related previous entries which had similar mood scores as a tool for inspiration.

## Style Guidelines:

- Primary color: Dark Slate Gray (#2F4F4F) to evoke calmness and introspection.
- Background color: Dark Gray (#333333) for a dark, neutral backdrop.
- Accent color: Light Steel Blue (#B0C4DE) for highlighting interactive elements.
- Body and headline font: 'Lora', a serif font for a classic and calming look, optimized for dark themes.
- Use simple, line-based icons to represent different moods and gratitude themes, with a light color for visibility on dark backgrounds.
- Maintain a clean and spacious layout for easy readability and comfortable reflection, optimized for dark themes.
- Subtle animations, like a gentle fade-in effect when displaying new entries, to create a soothing user experience on a dark background.
